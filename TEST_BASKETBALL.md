## 问题排查

### 如果命令不工作

1. **检查控制台错误**
   - 打开浏览器开发者工具（F12）
   - 查看 Console 标签是否有错误

2. **检查服务器状态**
   - 确认 `npm run dev` 正在运行
   - 访问 http://localhost:5173/

3. **清除缓存**
   - 刷新页面（Ctrl+F5 或 Cmd+Shift+R）
   - 清除浏览器缓存

### 如果统计不正确

1. **重置游戏**
   ```bash
   basketball reset
   ```

2. **重新测试**
   - 从头开始测试流程
