# 🏀 篮球游戏使用指南

## 游戏简介

这是一个集成在终端中的篮球小游戏，支持投篮、扣篮、传球等操作，具有连击系统和详细的统计功能。

---

## 🎮 游戏命令

### 1. 查看游戏界面
```bash
basketball
```
显示篮球场和游戏说明。

### 2. 投篮（Shoot）
```bash
basketball shoot
```
- **得分：** 2分
- **基础命中率：** 70%
- **连击加成：** 每次连击+2%命中率（最多+20%）
- **成功后：** 连击+1

### 3. 扣篮（Dunk）
```bash
basketball dunk
```
- **得分：** 3分
- **基础命中率：** 50%（难度更高）
- **连击加成：** 每次连击+3%命中率（最多+25%）
- **成功后：** 连击+2（扣篮奖励更多）

### 4. 传球（Pass）
```bash
basketball pass
```
- **成功率：** 80%
- **助攻概率：** 50%（传球成功后）
- **助攻得分：** 1分

### 5. 查看统计
```bash
basketball stats
```
显示详细的游戏统计数据：
- 总体数据（得分、命中率、连击）
- 投篮数据（次数、命中、命中率）
- 扣篮数据（次数、成功、命中率）
- 传球数据（次数、助攻、助攻率）
- 最近10条游戏记录

### 6. 重置游戏
```bash
basketball reset
```
清空所有数据，重新开始游戏。

## 🎯 游戏机制

### 连击系统

连击是游戏的核心机制：

1. **投篮成功** → 连击+1
2. **扣篮成功** → 连击+2
3. **投篮/扣篮失败** → 连击归零
4. **连击加成：**
   - 投篮：每次连击+2%命中率（最多+20%）
   - 扣篮：每次连击+3%命中率（最多+25%）

**示例：**
- 连击0次：投篮命中率70%
- 连击3次：投篮命中率76%（70% + 3×2%）
- 连击10次：投篮命中率90%（70% + 20%上限）

### 得分系统

- **投篮成功：** +2分
- **扣篮成功：** +3分
- **助攻：** +1分

### 命中率评级

统计界面会根据命中率显示不同颜色：

- **优秀（绿色）：** ≥70%
- **良好（青色）：** 50-69%
- **一般（黄色）：** 30-49%
- **较差（红色）：** <30%

## 📊 统计数据说明

### 总体数据
- **总得分：** 所有得分的总和
- **总命中率：** (投篮命中+扣篮成功) / (投篮次数+扣篮次数)
- **当前连击：** 当前连续命中次数
- **最高连击：** 历史最高连击记录

### 投篮数据
- **投篮次数：** 使用 `basketball shoot` 的次数
- **投篮命中：** 投篮成功的次数
- **投篮命中率：** 投篮命中 / 投篮次数

### 扣篮数据
- **扣篮次数：** 使用 `basketball dunk` 的次数
- **扣篮成功：** 扣篮成功的次数
- **扣篮命中率：** 扣篮成功 / 扣篮次数

### 传球数据
- **传球次数：** 使用 `basketball pass` 的次数
- **助攻次数：** 传球成功且队友得分的次数
- **助攻率：** 助攻次数 / 传球次数

## 🎲 游戏策略

### 新手策略
1. **多投篮：** 投篮命中率高（70%），适合积累连击
2. **保持连击：** 连击越高，命中率越高
3. **谨慎扣篮：** 扣篮难度大，失败会清空连击

### 进阶策略
1. **连击后扣篮：** 积累3-5次连击后，命中率提升，此时扣篮更安全
2. **混合战术：** 投篮+传球+扣篮，保持游戏多样性
3. **追求高分：** 连击10次后，投篮命中率达90%，扣篮命中率达75%

### 高分技巧
1. **建立连击：** 先用投篮建立5次以上连击
2. **扣篮得分：** 连击高时使用扣篮获得3分
3. **保持手感：** 避免连续扣篮失败，适时用投篮稳定连击

## 💡 开发说明

### 技术实现

**状态管理：** Pinia Store (`src/stores/basketball.js`)
- 管理游戏状态、得分、统计等数据
- 提供 shoot、dunk、pass 等方法

**命令系统：** 扩展支持子命令
- `basketball` → 显示游戏界面
- `basketball shoot` → 执行投篮
- `basketball stats` → 显示统计组件

**组件：**
- `BasketballCommand.vue` - 游戏主界面
- `BasketballStatsCommand.vue` - 统计界面

### 扩展方向

1. **更多游戏模式**
   - 计时模式（60秒内得分）
   - 挑战模式（达到目标分数）

2. **更多动作**

   - 抢断

3. **多人功能**
   - 排行榜
   - 对战模式
   - 好友系统

4. **视觉增强**
   - 更丰富的ASCII动画
